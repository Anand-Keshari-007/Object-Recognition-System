open the git bash at location hich you want to link with github
1.	git config --global user.name "your name"
2.	git config --global user.email "your email"
3.	touch .gitignore
4.	git init
5.	git add .
6.	git commit -m "Initial commit"
7.	git remote add origin (your github repository path) //make sure you clicked on SSH 	after creating the repo.

** NOW Create an SSH key for your github **

8.	ssh-keygen -t ed25519 -C "your email address"

9.	cat (path where the key is stored)
10. 	previous command will show your SSH key
11.	copy it and add it in your github.com>setting>SSH and GGH key>new SSH key.
12.	git push origin master

** Your project is now pushed onto your github repository **
